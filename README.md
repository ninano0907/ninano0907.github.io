# 소원의 행성

### Planet Wish — 매일 하늘이 전하는 오늘의 운세

당신의 생년월일을 바탕으로, 실제 천체의 위치를 계산해 만드는 특별한 운세 앱입니다.
막연한 랜덤 운세가 아니라 USNO 천문 모델을 기반으로 그날의 실제 행성 배치를 계산하고,
그 결과를 바탕으로 **사랑 · 재물 · 건강 · 직업**, 네 가지 영역의 이야기를 들려드립니다.

매일 똑같은 문구가 아니라, 하루하루 달라지는 하늘의 흐름을 그대로 담아냅니다.

## 어떻게 이용하나요

1. **생년월일 입력** — 처음 한 번만 입력하면 끝. 이후에는 바로 오늘의 운세로 이동합니다.
2. **수정구슬 선택** — 마음에 드는 구슬 하나를 골라 오늘의 기운을 불러옵니다.
3. **운세 확인** — 사랑 · 재물 · 건강 · 직업, 네 영역의 이야기가 펼쳐집니다.
4. **위성 소환** — 확인이 끝나면 위성이 하나 소환되어 내 행성 곁에 자리를 잡습니다.

## 4가지 영역

| 영역 | 내용 |
| --- | --- |
| 사랑 | 인연과 관계의 흐름 |
| 재물 | 돈과 기회에 대한 감각 |
| 건강 | 컨디션과 몸의 신호 |
| 직업 | 일과 성취에 대한 방향 |

## 나만의 행성계, 위성 시스템

한 번 소환된 위성은 사라지지 않습니다. 내 행성 주위를 돌며 하나씩 쌓여, 방문할 때마다
조금씩 더 풍성한 행성계를 이룹니다. 오늘 하루의 운세를 확인하는 것이 곧 나만의 하늘을
가꾸는 일이 되는 셈입니다.

며칠을 빠짐없이 들여다봤는지, 내 행성 곁에 위성이 몇 개나 모였는지 — 그 자체가 하나의
기록이자 즐거움이 됩니다.

## 왜 다른 운세 앱과 다른가요

- **근거 있는 계산** — 감으로 지어낸 문구가 아니라 실제 천체 위치 계산에 기반합니다.
- **매일 달라지는 이야기** — 그날그날의 행성 배치가 다르니, 내용도 매일 새롭습니다.
- **쌓이는 재미** — 결과를 보고 끝나는 게 아니라, 위성이라는 형태로 흔적이 남습니다.
